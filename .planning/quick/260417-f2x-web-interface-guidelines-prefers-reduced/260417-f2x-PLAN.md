**1. app/globals.css** — 파일 끝에 전역 규칙 추가:

```css
/* Accessibility: respect user motion preferences */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

이 전역 규칙이 모든 컴포넌트의 inline style 애니메이션에도 적용된다. 단, inline `style={{ animation: ... }}`은 CSS 캐스케이드 우선순위 때문에 `*` 선택자가 덮어쓰지 못할 수 있으므로, 아래 컴포넌트별 수정에서 추가로 처리한다.

   MiniSlotMachine 컴포넌트에 `useReducedMotion` 훅을 추가하되, 외부 라이브러리 없이 직접 구현:
   
   파일 상단 import 구역에 추가 (useState, useEffect 이미 있음):
   ```tsx
   // useReducedMotion 훅 추가 — 파일 내 선언
   function useReducedMotion() {
     const [reduced, setReduced] = useState(false)
     useEffect(() => {
       const mq = window.matchMedia('(prefers-reduced-motion: reduce)')
       setReduced(mq.matches)
       const handler = (e: MediaQueryListEvent) => setReduced(e.matches)
       mq.addEventListener('change', handler)
       return () => mq.removeEventListener('change', handler)
     }, [])
     return reduced
   }
   ```
   
   MiniSlotMachine 함수 본문 상단에 추가:
   ```tsx
   const reducedMotion = useReducedMotion()
   ```

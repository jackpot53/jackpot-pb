# Phase 4 — UI Design Contract

> Visual and interaction contract for Phase 4: History & Charts.
> Generated by gsd-ui-researcher, verified by gsd-ui-checker.
>
> Design system is fully established from Phase 3. This contract inherits all base tokens
> and extends only with AreaChart-specific color and the /charts page layout.

---

| Property | Value | Source |

## Spacing Scale

Identical to Phase 3 — no new exceptions for Phase 4.

| Token | Value | Tailwind | Usage |
|-------|-------|----------|-------|
| xs | 4px | gap-1 / p-1 | Icon gaps, inline badge padding |
| sm | 8px | gap-2 / p-2 | Compact element spacing |
| md | 16px | gap-4 / p-4 | Default card padding, tab spacing |

## Component Specification

### Charts Page Tab Structure

```
Tabs (defaultValue="annual", className="w-full")
  TabsList
    TabsTrigger value="annual"  label: "연간"
    TabsTrigger value="monthly" label: "월간"
  TabsContent value="annual"
    <AnnualReturnChart data={annualData} />
  TabsContent value="monthly"
    <MonthlyPortfolioChart data={monthlyData} />
```

shadcn Tabs component (`components/ui/tabs.tsx`) — already installed.

### Annual Return Chart Card (`components/app/annual-return-chart.tsx`)

New component. Wraps Card + recharts AreaChart for year-over-year return data.

```
Card
  CardHeader
    CardTitle: "연간 수익률" (20px semibold)
    CardDescription: "연도별 전체 자산 수익률 (%)" (14px muted)
  CardContent
    IF data.length < 2:
      → render <InsufficientDataMessage />
    ELSE:
      ResponsiveContainer width="100%" height={320}
        AreaChart data={annualData} margin={{ top: 8, right: 16, bottom: 0, left: 16 }}
          defs
            linearGradient id="areaGradient"
              stop offset="5%" stopColor="#3B82F6" stopOpacity={0.3}
              stop offset="95%" stopColor="#3B82F6" stopOpacity={0}
          CartesianGrid strokeDasharray="3 3" stroke="var(--border)"
          XAxis
            dataKey="year"  (number: 2023, 2024, 2025...)
            tickFormatter={(v) => String(v)}
            tick={{ fontSize: 12, fill: "var(--muted-foreground)" }}
          YAxis
            tickFormatter={(v) => `${v > 0 ? '+' : ''}${v.toFixed(1)}%`}
            tick={{ fontSize: 12, fill: "var(--muted-foreground)" }}
            domain={['auto', 'auto']}
          Tooltip content={<AnnualTooltip />}
          Area
            type="monotone"
            dataKey="returnPct"     (float: YoY return %)
            stroke="#3B82F6"
            strokeWidth={2}
            fill="url(#areaGradient)"
            dot={{ r: 4, fill: "#3B82F6", strokeWidth: 0 }}
            activeDot={{ r: 6 }}
```

Data shape per point:
```ts
interface AnnualDataPoint {
  year: number           // e.g. 2024
  returnPct: number      // e.g. 12.5 (meaning +12.5%)
  totalValueKrw: number  // snapshot totalValueKrw at year-end — for tooltip only
}
```

### AnnualTooltip (custom recharts tooltip)

Rendered inside the AreaChart as `<Tooltip content={<AnnualTooltip />} />`.

```
div (bg-card border border-border rounded-md px-3 py-2 shadow-sm text-sm)
  p: "{year}년" — 14px normal foreground
  p: "{returnPct formatted as +N.N% / -N.N%}" — 14px semibold
     → text-emerald-600 if positive, text-red-600 if negative
  p: "자산 총액 ₩{totalValueKrw formatted}" — 12px muted-foreground
```

### Monthly Portfolio Chart Card (`components/app/monthly-portfolio-chart.tsx`)

New component. Wraps Card + recharts AreaChart for 12-month rolling portfolio value.

```
Card
  CardHeader
    CardTitle: "월별 자산 총액" (20px semibold)
    CardDescription: "최근 12개월 포트폴리오 총액 (KRW)" (14px muted)
  CardContent
    IF data.length < 2:
      → render <InsufficientDataMessage />
    ELSE:
      ResponsiveContainer width="100%" height={320}
        AreaChart data={monthlyData} margin={{ top: 8, right: 16, bottom: 0, left: 16 }}
          defs
            linearGradient id="monthlyGradient"
              stop offset="5%" stopColor="#3B82F6" stopOpacity={0.3}
              stop offset="95%" stopColor="#3B82F6" stopOpacity={0}
          CartesianGrid strokeDasharray="3 3" stroke="var(--border)"
          XAxis
            dataKey="label"   (string: "2025-04", display as "4월" or "25.04")
            tickFormatter={(v) => formatMonthLabel(v)}  // "YYYY-MM" → "YY.MM"
            tick={{ fontSize: 12, fill: "var(--muted-foreground)" }}
          YAxis
            tickFormatter={(v) => formatKrwCompact(v)}  // e.g. ₩1.2억 / ₩500만
            tick={{ fontSize: 12, fill: "var(--muted-foreground)" }}
            domain={['auto', 'auto']}
            width={72}  // wider to fit KRW compact labels
          Tooltip content={<MonthlyTooltip />}
          Area
            type="monotone"
            dataKey="totalValueKrw"
            stroke="#3B82F6"
            strokeWidth={2}
            fill="url(#monthlyGradient)"
            dot={{ r: 4, fill: "#3B82F6", strokeWidth: 0 }}
            activeDot={{ r: 6 }}
```

Data shape per point:
```ts
interface MonthlyDataPoint {
  label: string          // "YYYY-MM" e.g. "2025-04"
  totalValueKrw: number  // snapshot totalValueKrw
  returnPct?: number     // optional: MoM return % — for tooltip only
}
```

Y-axis compact formatter (`formatKrwCompact`):
- Values >= 100,000,000: display as "₩{N.N}억"
- Values >= 10,000,000: display as "₩{N}천만"
- Values >= 1,000,000: display as "₩{N}백만"
- Others: "₩{N,NNN}" (full integer)

### MonthlyTooltip (custom recharts tooltip)

```
div (bg-card border border-border rounded-md px-3 py-2 shadow-sm text-sm)
  p: "{label formatted as YYYY년 M월}" — 14px normal foreground
  p: "₩{totalValueKrw formatted with Intl.NumberFormat ko-KR}" — 14px semibold foreground
  p (conditional, if returnPct present):
    "전월 대비 {returnPct formatted as +N.N% / -N.N%}" — 12px
    → text-emerald-600 if positive, text-red-600 if negative
```

### InsufficientDataMessage (shared empty/insufficient state)

Shared between both chart cards. Rendered when `data.length < 2`.

Two variants based on context:

**Variant A — No snapshots at all (data.length === 0):**
```
div (flex flex-col items-center justify-center h-[320px] gap-3)
  p (16px semibold foreground): "데이터를 수집하고 있습니다"
  p (14px muted-foreground):
    "{firstSnapshotDate}부터 데이터를 수집하고 있습니다. 내일 다시 확인해주세요."
    → if firstSnapshotDate not yet known (truly 0 snapshots):
      "첫 번째 스냅숏이 오늘 자정(KST)에 기록됩니다."
```

**Variant B — Exactly 1 snapshot (data.length === 1):**
```
div (flex flex-col items-center justify-center h-[320px] gap-3)
  p (16px semibold foreground): "데이터가 충분하지 않습니다"
  p (14px muted-foreground): "차트를 표시하려면 최소 2일의 데이터가 필요합니다."
```

Both variants: no CTA button (there is nothing for the user to do — cron handles it automatically).

## Loading States

Charts page is a Server Component. Data is fetched server-side from the snapshot table.
No client-side loading skeletons are required for the charts themselves.

| Scenario | Treatment |
|----------|-----------|
| Page initial load | Next.js Suspense boundary with skeleton card |
| Suspense fallback | `<Skeleton className="h-[400px] w-full rounded-xl" />` inside a Card |
| No snapshot data | `<InsufficientDataMessage>` (Variant A) inside chart CardContent |
| 1 snapshot only | `<InsufficientDataMessage>` (Variant B) inside chart CardContent |
| Data fetch error | Error card — see Copywriting Contract |

Suspense wrapper:
```tsx
<Suspense fallback={<ChartPageSkeleton />}>
  <ChartsPageContent />
</Suspense>
```

`ChartPageSkeleton`: full-width Card with `<Skeleton className="h-[400px] w-full" />` inside CardContent.

Inherits all formatters from Phase 3 (`lib/portfolio.ts`). Phase 4 adds two new formatters:

| Formatter | Input | Output example | Notes |

## Interaction Contract

| Interaction | Behavior |
|-------------|----------|
| Tab switch ("연간" / "월간") | Client-side tab swap, no page navigation, no data re-fetch |

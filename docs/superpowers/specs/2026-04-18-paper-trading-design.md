# 모의투자 백테스팅 페이지 설계

## 개요

로보어드바이저 추천 포트폴리오들의 과거 성과를 시뮬레이션하는 백테스팅 페이지. 사용자가 기간을 선택하고 여러 메트릭으로 전략의 역사적 성과를 분석할 수 있다.

## 핵심 요구사항

| 항목 | 요구사항 |
|------|---------|
| **네비게이션** | 사이드바에 "모의투자" 메뉴 추가 (로보어드바이저 아래) |
| **대상** | 로보어드바이저 추천 포트폴리오들 |
| **데이터** | 기존 `backtest_results` 테이블 활용 |
| **초기자금** | 고정값 (₩100,000,000) |
| **리밸런싱** | UI에서 옵션 제시하되, 향후 개선 대상 |

## 사용자 플로우

1. 사이드바에서 "모의투자" 클릭
2. `/paper-trading` 페이지 진입
3. 기간 선택 (버튼: 1년/3년/5년/전체 또는 날짜 피커)
4. 선택한 기간의 백테스트 결과 조회 및 시각화
5. 각 메트릭을 통해 전략 성과 분석

## 페이지 레이아웃

### 상단 제어판
```
┌─────────────────────────────────────────────┐
│ 모의투자 백테스팅                           │
├─────────────────────────────────────────────┤
│ [1년] [3년] [5년] [전체]  📅 [날짜 범위 선택] │
│ 초기자금: ₩100,000,000                       │
└─────────────────────────────────────────────┘
```

### 메인 콘텐츠

#### 1. 누적 수익 차트
- **타입**: 라인 차트 (Recharts)
- **표시**: 시간 흐름에 따른 자산 변화 (로봇별 다중선)
- **Y축**: 누적 자산값 (₩)
- **범례**: 각 로봇별 색상 구분

#### 2. 핵심 메트릭 카드
3개 카드로 구성:
- **총 수익률**: 기간 동안의 누적 수익률 (%)
- **샤프 비율**: 위험 대비 수익 효율성
- **MDD (최대낙폭)**: 최악의 경우 낙폭 (%)

#### 3. 연도별 비교 테이블
- **행**: 각 로보어드바이저 전략
- **열**: 각 연도 + 전체 평균
- **셀**: 해당 연도의 수익률 (%)
- **컬러**: 양수는 녹색, 음수는 빨간색

#### 4. 월별 리턴 히트맵
- **행**: 월 (1월~12월)
- **열**: 연도
- **셀**: 월간 수익률 (색상 강도)
- **범례**: -5% ~ +5% 정도 범위

## 데이터 흐름

1. **조회**: `backtest_results` 테이블에서 선택 기간의 일일 데이터 조회
2. **필터링**: 사용자 선택 기간(1년/3년/5년/전체 또는 날짜 범위)으로 필터
3. **계산**:
   - 수익률 = (최종자산 - 초기자금) / 초기자금 × 100
   - 샤프 비율 = 연간 수익률 / 일일 수익률 표준편차 × √252
   - MDD = 누적 최고점에서 이후 최저점까지의 낙폭 퍼센트
4. **렌더링**: 차트, 테이블, 히트맵 생성

## UI/UX 상세

### 기간 선택
- **버튼**: "1년", "3년", "5년", "전체" (Tailwind 버튼 그룹)
- **피커**: 캘린더 아이콘 클릭 시 날짜 범위 선택 (shadcn/ui Dialog + 수동 날짜 입력 또는 간단한 캘린더)
- **기본값**: "1년" 선택 상태

### 차트 상호작용
- 마우스 호버 시 해당 날짜의 정확한 값 툴팁 표시
- 범례 클릭으로 특정 로봇 선택/해제 가능

### 반응형
- 모바일: 차트는 스크롤 가능한 수평 스크롤로
- 테이블/히트맵: 작은 화면에서도 가독성 유지

## 제약사항 & 향후 개선

- **리밸런싱**: 현재 고정된 결과만 표시, 향후 온디맨드 계산 지원 고려
- **초기자금**: 고정값으로 시작, 나중에 커스터마이징 가능
- **성능**: 매우 긴 기간 조회 시 차트 데이터 포인트 집계 필요 (월별 집계 등)
